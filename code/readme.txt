编译环境：
Windows11
Qt Creator 4.11.1
Based on Qt 5.14.1 (MSVC 2017, 32 bit)
编译说明：
1. 下载并安装QT Creator IDE；
2. 打开Qt Creator，选择File -> Open File Or Project
3. 选择同路径下的byyl-combined.pro文件,即可使用Qt Creator打开项目
4. 在左下角选择 build工具 -> 配置 MSVC 2017, 32 bit
5. 点击左下角绿色箭头的运行按钮即可编译完成并运行。

提示：不建议使用命令行进行编译，可能会无法解决的问题，目前本人也尚未找到解决方法。
